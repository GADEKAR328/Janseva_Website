# Janseva Vidyalaya, Wadzire — School Website 🎓

A fully responsive website built for **श्रीमती स. गं. लंके माध्यमिक विद्यालय व आनंदीबाई ना. एरंडे उच्च माध्यमिक विद्यालय** (Janseva Vidyalaya), a secondary and higher secondary school established in 1958, located in Wadzire, Parner Taluka, Ahilyanagar District, Maharashtra, India.

This project was built as a self-initiated portfolio project to showcase front-end web development and UI/UX design skills — while giving a real educational institution a modern digital presence.

🔗 **Live Site:** [gadekar328.github.io/Janseva_Website](https://gadekar328.github.io/Janseva_Website/)](https://gadekar328.github.io/Janseva_Website/)

---

## 📌 About the Project

The school did not have any online presence before this project. The goal was to design and build a complete, content-rich, mobile-friendly website — from information architecture to deployment — that reflects the school's history, values, and day-to-day life, while making it easy for parents and prospective students to learn about admissions.

## ✨ Features

- 🏫 **About the School** — history, values, and background since 1958
- 🎯 **Vision & Inspiration** — guiding role models and school philosophy
- 🗣️ **Principal's Message**
- 📚 **Academics** — curriculum details for Secondary (9th–10th) & Higher Secondary (11th–12th)
- 📝 **Admissions** — step-by-step admission process with an online inquiry form
- 🏗️ **Facilities & Infrastructure** — labs, library, playground, and more, presented via an at-a-glance data section
- 🖼️ **Photo Gallery** — school events, activities, and campus life
- 🎓 **Alumni Spotlight** — showcasing alumni across corporate, government, IT, healthcare, research, and education sectors
- 📰 **Events & News** — latest school updates
- 👩‍🏫 **Teacher Directory**
- 📍 **Contact Section** — with embedded Google Map, phone, and email
- 🔗 **Social Links** — Facebook, YouTube, Instagram

## 🚀 Getting Started

To run this project locally:

```bash
# Clone the repository
git clone https://github.com/GADEKAR328/Janseva_Website.git

# Navigate into the project folder
cd Janseva_Website

# Open index.html in your browser
```

No build tools or dependencies required — it's a static site.
